# Cost-Benefit-Analysis-Hecate-Offshore-Wind-Farm-
This CBA evaluates the economic and social impacts of the proposed 400 MW Hecate Offshore Wind Farm in British Columbia. Using a 10-step framework, it weighs capital, operational, and terminal costs against clean energy benefits like carbon offsets. Includes an onshore alternative comparison, sensitivity modeling, and NPV/LCOE financial models.
